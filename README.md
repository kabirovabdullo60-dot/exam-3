# 🚀 Платформаи Муосири Вакансияҳо ва Маркази Алоқа (CRM Suite)

Ин лоиҳа маҷмӯи саҳифаҳои адаптивӣ (Responsive) ва муосири веб мебошад, ки барои платформаҳои корёбӣ, саҳифаҳои касбӣ (Careers) ва системаҳои алоқаи тиҷоратӣ (CRM) пешбинӣ шудааст. Танзимоти UI комилан дар услуби муосири **Dark Mode** бо истифода аз **Tailwind CSS** сохта шудааст.

---

## 🗺️ Шарҳи Умумии Саҳифаҳо

Лоиҳа аз чор қисми асосии модулӣ иборат аст, ки раванди пурраи корбарро аз шиносоӣ то супоридани ариза фаро мегирад:

### 1. Баннери Асосии Намоишӣ (Hero Spotlight)
* **Эзоҳ:** Саҳифаи аввали вебсайт бо заминаи суратӣ.
* **Имкониятҳо:** Саҳифаи дорои қабати ториккунанда барои хондани осони матнҳо ва тугмаи даъват ба амал (CTA).

### 2. Саҳифаи Тавсифи Вакансия (Job Description)
* **Эзоҳ:** Саҳифаи тақсимшуда ба ду сутун барои вакансияи Web Designer.
* **Имкониятҳо:** Меъмории адаптивӣ. Дар телефонҳо корти кӯтоҳи маълумот (маош, таҷриба, мӯҳлат) ба аввал мегузарад, то корбар маҷбур нашавад тамоми матни дарозро скрoлл кунад.

### 3. Портали Аризасупорӣ (Application Form)
* **Эзоҳ:** Формаи электронии қабули ҳуҷҷатҳо.
* **Имкониятҳо:** Дорои бахши махсуси боркунии файлҳо (Drag & Drop), инпути рақами телефон бо рамзи давлат ва блоки амниятии ҳифз аз ботҳо (Капча).

### 4. Маркази Алоқа ва Бронкунии Вохӯриҳо (Contact Hub)
* **Эзоҳ:** Саҳифаи тамос ва алоқаи муштариён.
* **Имкониятҳо:** Тарҳи касбӣ барои нишон додани суроға, телефон, email ва блоки алоҳида барои фармоиш додани машваратҳои ройгон.

---

## ⚡ Технологияҳо ва Оптимизатсия

* **HTML5** - Сохтори семантикии саҳифаҳо барои амният ва дастрасии баланд.
* **Tailwind CSS (v4)** - Барои тарҳрезӣ ва адаптивияти зуд бидуни вазнин кардани коди сайт.
* **Векторҳои ичкии SVG** - Истифодаи иконаҳои сабук ва босифат, ки дар дилхоҳ экран хира намешаванд.

### Навовариҳои асосӣ дар ин код:
* **Шрифтҳои Свежий:** Андозаи матнҳо вобаста ба экрани телефон ва компютер ба таври худкор тағйир меёбанд.
* **Эффектҳои Ҳаракат:** Ҳангоми бурдан ё пахш кардани тугмаҳо ва инпутҳо, эффектҳои ҳамвор нишон дода мешаванд.
* **Рангҳои Роҳат (Dark Mode):** Истифодаи рангҳои торики махсус барои он ки чашми корбар ҳангоми хондани матнҳои дароз хаста нашавад.

---

## 🛠️ Чӣ тавр истифода барем?

1.  Файлҳои кодро дар папкаи лоиҳаи худ ҷойгир кунед.
2.  Аксҳои заруриро дар ҳамон папка (директория) гузоред.
3.  Файлҳои `.html`-ро мустақиман дар дилхоҳ браузер (Chrome, Safari, Firefox) кушоед ё тавассути Live Server иҷро кунед.
